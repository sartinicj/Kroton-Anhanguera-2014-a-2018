Ultrasurf is a product of Ultrareach Internet Corporation. II. Installation and Usage

1. Download and Installation

UltraSurf does not require installation. You can start the program simply by double-clicking the executable file. 

 

Users may save UltraSurf to a flash drive or access it from their email. The program may be used on any computer and will leave no trace when deleted.

 

The UltraSurf icon is a sailboat navigating around the globe. Users may change the file name for their own safety and convenience.

 

2. Start UltraSurf

 

After you double click the executable file, UltraSurf will open the Interface window. The interface displays the function keys and the operation status.

 

The UltraSurf temporary file folder utmp and system initialization file u.ini will be put under the same location as the exe file (Version 10.00 and higher).

 

3. Server Connection and Operation Status Display

 

The UltraSurf interface displays the status of server connection and operation. At startup, UltraSurf automatically searches for and chooses the highest speed server available. The check box and the green light indicate which server is currently in use. You may select a different server by checking another box if the current one is not working properly.

 

Connection speed is an indicator of the performance of the server: 100% is the highest speed and is equivalent to a direct connection.

 

4. How to Use the UltraSurf Program

 

By using Internet Explorer (IE), UltraSurf users are able to browse any website and open new windows.

 

You may click "Home" butto to start browser. Or you may enter any URL in the small URL window of the IE, UltraSurf will bring up the web site immediately.

 

You may minimize the interface window when using UltraSurf, you may hide it in system tray by closing the interface window, or you may hide it completely in background by Atl+Ctl+H. You may restore the interface by Alt+Ctl+S. 

 

UltraSurf automatically use Explorer (IE), but the new Google Chome and Firefox browsers also uses system proxy setting and they will also work with Ultrasurf by default. You can test it by entering: http://formyip.com, if you see the IP of "65.49.14.X", you are protected by UltraSurf. Note that "Clear History" "Clear Cookies" and "Clear Cache" functions only work for IE, if you use other browsers, you need to clean the records by yourself or using other tools.

 

5. Exiting the UltraSurf Program

 

You can click the “Exit” key on the UltraSurf  to exit the Ultra.

 

Once you exit UltraSurf, your IE browser is no longer protected by UltraSurf. For your safety, during the 20 second count-down before exiting, an “Exit Warning” will be appear with the following options:

 

·   Close IE and exit.

This option allows you to close all current browser windows when you exit UltraSurf. This option is recommended as it is very likely that the browser’s windows stored the URLs of some of the censored websites that you visited while using UltraSurf.

 

·   Keep IE and exit.

If you have a special need to keep your current IE browser’s windows open, and you are aware of the safety issues, you can select this option, which prevents UltraSurf from closing any of the browser’s windows which are open during the exiting process.

7. Advance Settings

 

The default setting of UltraSurf is intended to protect the users’ safety. Users with special needs may alter these settings to specify whether they want to clear the browsing history, delete cookies or use proxies. The following are detailed instructions for the options in “advanced settings”.

1       “Enable HotKeys”: when this option is checked, the HotKeys will be enabled; if this option is unchecked, HotKeys will be disabled. You may want to disable this feature if you do not use it or the hot keys are conflict with other programs.

2       “Hide Windows”: when this option is checked, UltraSurf wil start in background. To bring it to foreground, use Ctl+Alt+S.

3       “Do not use IE”: users who do not want to use IE with UltraSurf may select this option. When this option is checked, UltraSurf will not manage or change any IE settings and other IE related options will be inactive.

4        “Start IE automatically”: when this option is checked, UltraSurf will automatically open an IE browser upon startup; if this option is unchecked, UltraSurf will not open an IE browser or any other website when started.

5       “Delete cookies automatically upon exiting”: when this option is checked, UltraSurf will automatically delete cookies when exiting (this is the default setting). Users who have decided not to delete cookies should uncheck this option.

6       “Delete history automatically upon exiting”: when this option is checked, UltraSurf will automatically delete the browsing history when exiting.

7        “Close IE immediately upon exiting”: when this option is checked, UltraSurf will close all IE browsers and all websites when exiting. If this option is unchecked, users may choose to retain or manually close IE windows (refer to Step 5 - Exit UltraSurf Program for details).

8       “Do not close IE upon exiting”: when this option is checked, UltraSurf will not close IE windows, nor will it show the “Exit Warning” window when exiting.

9       “Hide Golden Lock (requires UltraSurf to be restarted in order to take effect)”: by default, a Golden Lock on lower right corner of the window indicates that you being protected by UltraSurf. You may hide the Golden Lock by checking this option which will take effect the next time you start UltraSurf.

10  “Proxy setting” button: enters the proxy setting window (refer to “Manual Proxy Setting” below).

11  “Delete Cookie” button: deletes all the cookies immediately.

12  “Delete History” button: deletes the entire browsing history immediately

13  “Local Listening Port (26-65535): you can set local listening port from 26 to 65535, default 9666.

14  “Reset to Default Settings” button: reset all settings to default value, a window will pop up, hit “Yes” to confirm, “No” to cancel. Then hit “OK” to take effect, “Cancel” to cancel.

 

8. Manual Proxy Setting

 

Normally users do not need to enter proxy setting themselves. By default UltraSurf will choose the proxy setting based on your IE proxy setting.

 

In certain cases users may need to use a proxy to access Internet. This option allows users to manually set a proxy for UltraSurf.

 

The following is the instruction for proxy setting:

 

·   “Auto-Detect Proxy”: when this option is checked, UltraSurf will automatically detect the proxy setting of IE browser.

·   “Manual Proxy Setting”: when this option is checked, you can enter your own proxy setting.

·   “Directly use UltraSurf”: when this option is checked, users will use 100% transparent and highly encrypted UltraSurf directly, without the involvement of other proxy servers. If the user has entered a proxy server on IE, that proxy server will not be used; it will resume being the proxy server for IE after you exit the UltraSurf program.

 

III. Functional Specification of UltraSurf

 

(1)   Privacy Protection

 

UltraSurf protects users’ Internet privacy through anonymous surfing and browsing: the program hides IP addresses and locations, cleans browsing history, cookies and more.

 

(2)   Strong Encryption and High Security

 

UltraSurf uses industry standard strong encryption SSL, which is used for financial data transfers such as online banking and shopping. All data which is transmitted through UltraSurf is highly encrypted, ensuring that nobody other than the user will ever see the content.

 

      (3) Supports HTTPS Tunneling

 

UltraSurf supports https tunneling. Users can visit “https” websites through UltraSurf. Because the content of “https” websites is already encrypted, data from these websites will be doubly protected through UltraSurf.

 

 (4) Supports All HTTP Based Protocols

 

UltraSurf supports all http based protocols, including webmail, http based data streaming, javascripts and DHTML, etc – all email sent through UltraSurf is highly encrypted.

 

        (5) Supports Secure Data Uploading and Downloading

 

UltraSurf supports both uploading and downloading by highly encrypting the data.

5. Q:  What should I do if UltraSurf cannot connect to the Internet?

 A:  Please follow the steps below:
1) Verify that you can connect directly to the Internet.
2) If you are able to connect to the Internet, shutdown your firewall or anti-virus software. Then restart the UltraSurf.
3) If you still cannot connect to the Internet, it is probably because our servers are too busy – please try again later. If this is a recurring problem, please send us the feedback by clicking "Feedback" button on the lower right corner and copy the encrypted message and send to us in email to info8@ultrasurf.us, along with your detailed platform and network environment information, so that we can provide you with further assistance.

 

6. Q:  Why is UltraSurf occasionally unable find servers?

 A:  There may be a temporary issue with the network.  If so, you should try again later. If the UltraSurf user interface displays "network connection failure, check your network and proxy setting", please check the network connection setting or proxy setting, and firewall setting. If you still cannot connect, please send us the feedback by clicking "Feedback" button on the lower right corner and copy the encrypted message and send to us in email to info8@ultrasurf.us, along with your detailed platform and network environment information, so that we can provide you with further assistance.
